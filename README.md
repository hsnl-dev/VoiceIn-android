# Voice-in-android
聚寶盆 - (version: voiceIn)

Prequistie
===
- JDK >= 1.7
- Android studio
- Android SDK 
  - build tool 23.0.2

Development
===
- Download and install Android studo
- Import project : ```New > import project``` navigate to your floder

Build APK
===
```
cd your project folder
./gradle assembleRelease
```
